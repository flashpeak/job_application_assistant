Software Engineer Job Application

Job Description

We are seeking a talented React Native Developer to join our team at Meta. In this role, you will be responsible for building and maintaining high-performance mobile applications using React Native. You will work closely with our product and design teams to create seamless cross-platform experiences that millions of people use every day.

Key responsibilities:

  • Develop new features and maintain existing functionality in our React Native mobile applications
  • Collaborate with cross-functional teams to define, design, and ship new features
  • Ensure the performance, quality, and responsiveness of applications
  • Identify and correct bottlenecks and fix bugs
  • Help maintain code quality, organization, and automatization

Requirements:

  • 3+ years of experience with React Native development
  • Strong understanding of React and React Native frameworks
  • Experience with iOS and Android development
  • Proficiency in JavaScript/TypeScript
  • Experience with mobile UI design principles and patterns
  • Bachelor's degree in Computer Science or related field
